# Змейка (Snake)

Классическая игра «Змейка», реализованная на Python с использованием Tkinter.

## Особенности

- Управление стрелками или WASD
- Пауза (Пробел)
- Изменение скорости игры
- Изменение размера окна
- Адаптивный размер клеток при масштабировании
- Счётчик очков
- Сообщение о победе при заполнении поля

## Требования

- Python 3.8 или выше
- Tkinter (входит в стандартную поставку Python)

## Установка и запуск

1. Склонируйте репозиторий или скачайте архив с кодом.

2. Убедитесь, что у вас установлен Python:
   ```bash
   python --version
   
3. Перейдите в папку с проектом и запустите игру:
    ```python main.py```
